# TRABAJO-DE-ELECTRCIDAD-Y-MAGNETISMO
# Movimiento de un electrón en un campo eléctrico

Este programa simula el movimiento de un electrón dentro de un campo eléctrico uniforme.

## Datos

- Velocidad: 3 × 10^6 m/s
- Campo eléctrico: 400 N/C
- Tiempo: 0 a 3000 ns

## Ejecución

Instalar librerías:

pip install -r requirements.txt

Ejecutar:

python electron.py

## Resultados

Se generan:

- figura1.png → gráfica del movimiento
- animacion.gif → animación del electrón

## Descripción

El electrón presenta:

- Movimiento uniforme en x
- Movimiento acelerado en y

Esto genera una trayectoria parabólica.
